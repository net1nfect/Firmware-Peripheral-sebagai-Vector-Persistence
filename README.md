SEDANG DALAM PROSES PERKEMBANGAN PENELITIAN

noted
rencana gua :

· keyboard gaming dengan RGB lighting control yang bisa di-flash via software
· firmware modifikasi yang membuat keyboard terdeteksi sebagai HID composite device
